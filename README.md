# 💰 Expense Tracker (Streamlit + SQLite)

## 📌 Features
- Add & delete expenses
- Category-wise tracking
- Budget alerts
- Monthly analytics
- Pie & Bar charts

## 🛠 Tech Stack
- Python
- Streamlit
- SQLite
- Pandas
- Matplotlib

```bash
pip install -r requirements.txt
